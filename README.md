# 🚗 Car Rental System

## 📌 Description
The Car Rental System is an application that allows users to browse, book, and manage car rentals online. It automates vehicle availability, customer details, and rental transactions for a smooth and efficient experience.

---

## ✨ Features
- User registration and login  
- View available cars  
- Book and cancel rentals  
- Rental cost calculation  
- Admin can manage cars and bookings  

---

## 🛠️ Technologies Used
- Frontend: HTML, CSS, JavaScript  
- Backend: Java / Python / Node.js  
- Database: MySQL  

---

## ▶️ How to Run
1. Clone the repository  
2. Open project folder  
3. Run the main file / server  
4. Access in browser  

---

## 🎯 Objective
To simplify and automate the car rental process, reducing manual work and improving user experience.
